# ПАРА КАРТ

Карточная игра на память. Переворачивай карточки и находи одинаковые пары.

## Играть

[https://astapenko74.github.io/para-cart/](https://astapenko74.github.io/para-cart/)

## Особенности

- 8 пар карточек с эмодзи
- Звуковые эффекты при переворачивании и совпадении
- Анимированный главный экран
- Адаптивный дизайн (мобильные устройства и десктоп)
- PWA — можно добавить на домашний экран

## Технологии

HTML, CSS, JavaScript — без зависимостей и фреймворков.
